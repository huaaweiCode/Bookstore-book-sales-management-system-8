# 书店图书销售管理系统


## 1、项目介绍

书店图书销售管理系统8拥有两种角色

管理员：用户管理、图书管理、分类管理、订单管理、订单发货等

用户：登录注册、条件分类查询图书、购物车、订单记录、收货等


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql5.x
- maven版本：不需要
- 硬件环境：Windows 或者 Mac OS
